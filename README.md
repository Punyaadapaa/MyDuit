## ✨ Fitur Utama

- **💰 Dashboard Saldo Total**: Pantau sisa saldo kamu secara real-time berdasarkan riwayat transaksi.
- **📝 Catat Transaksi Baru**: Tambahkan data pemasukan atau pengeluaran baru lengkap dengan nominal dan keterangan.
- **📜 Riwayat Transaksi**: Lihat daftar riwayat keuanganmu. Pemasukan ditandai dengan warna hijau (+), dan pengeluaran dengan warna merah (-).
- **🎨 UI Modern & Clean**: Dibangun menggunakan Material Design 3 untuk tampilan yang minimalis dan memanjakan mata.

## 🛠️ Teknologi & Materi yang Digunakan

Proyek ini dibangun menggunakan teknologi pengembangan Android modern:

- **Bahasa Pemrograman**: [Kotlin](https://kotlinlang.org/)
- **UI Toolkit**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (Sistem UI deklaratif modern untuk Android)
- **Desain Sistem**: [Material Design 3](https://m3.material.io/)
- **Arsitektur**: State Management dasar bawaan Compose (`remember`, `mutableStateOf`, `mutableStateListOf`).
